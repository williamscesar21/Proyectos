Main Page - YouTube

HTML / CSS

Fecha: 08/08/2023

Puedes verlo en: https://github.com/williamscesar21/Youtube.git

Mi proyecto consiste en una imitación del frontend de la página de YouTube utilizando HTML y CSS. La interfaz de usuario se asemeja a la de YouTube, con un diseño responsive que se adapta a diferentes dispositivos. Incluye elementos familiares como la barra de búsqueda, la reproducción de videos en miniatura y la navegación por categorías. Los videos se pueden reproducir en pantalla completa, y se ha aplicado un estilo atractivo y coherente en todo el sitio para proporcionar una experiencia de usuario similar a la de YouTube. El proyecto se enfoca en la presentación y la apariencia visual del sitio, con el objetivo de brindar a los usuarios una sensación familiar al interactuar con él.
